# Glossary

## Autarkic

Self-powered or self-grounding in the sense used by this project: a system
should minimize appeals to opaque external authority when establishing trust in
its reasoning or substrate.

## Autarkic System

An integrated artificial system whose logic, implementation, and computational
substrate are exposed enough that it can maintain scoped confidence in itself.

## Autarkic Formal System

A formal system package that connects:

- an object language;
- a proof or refutation apparatus;
- an explicit encoding of syntax and proofs;
- a substrate transition model;
- executable artifacts that check the above.

An AFS is not merely a theorem prover. It must support the AS objective of
cognitive sovereignty by making the relation between formal confidence and
embodied computation inspectable.

## Cognitive Sovereignty

The project target in which an artificial entity can reason about itself, its
knowledge, and its computational substrate without treating hidden external
machinery as the final unquestioned authority.

## Formal Confidence

A scoped confidence claim made inside, or about, a formal system. In the SJAS
line of work this centers on consistency and self-provability of consistency
under carefully limited expressivity.

## Geometrically Explicit Logic Circuit

PRC's term for a circuit whose formal description includes the geometry and
routing of signal transport, instead of treating wires as an implementation
detail outside the logical model.

## Reconfiguration

A substrate-level change to the role, routing, memory, or future behavior of a
computational element. In AS, reconfiguration is only useful if the conditions
under which it preserves required guarantees can be stated and checked.

## Self-Justifying Axiom System

A formal system, in Willard's sense, that can retain consistency relative to
Peano Arithmetic while proving selected consistency statements about itself by
weakening or tuning language expressivity and deduction method.

## Universal Cell

PRC's primitive homogeneous computational element. The reviewed PRC material
models Universal Cells as explicit state machines with roles such as wire,
processor, and stem cell, and with fields for input, output, memory, control,
and buffering.
